# Contributing to NightCityVerse

Thank you for your interest in contributing! We appreciate any help to improve the project.

## How to contribute

1. Fork the repository.
2. Create a feature branch: `git checkout -b feature/my-feature`.
3. Commit your changes with clear messages.
4. Run tests: `npm test`.
5. Push your branch and open a pull request against `main`.

## Code style

- Prettier formatting should be preserved.
- Follow existing style in TypeScript files.
- Keep new dependencies minimal.

## Reporting bugs

- Use GitHub Issues.
- Include steps to reproduce, expected behavior, and actual behavior.

## Feature requests

- Open an issue with a clear summary and motivation.
- Include a proposed solution if possible.

## Security issues

- Do not use Issues for security vulnerabilities; use `SECURITY.md` guidance.
