# Security Policy

## Reporting a Vulnerability

If you discover a security issue, do not open a public issue. Instead, send an email to the maintainer at `erdpy@pm.me` with:

- A concise description of the vulnerability
- Steps to reproduce
- Impact assessment
- Suggested mitigation

## Supported Versions

NightCityVerse is maintained on the `main` branch and receives security updates there.

## Response Process

1. Acknowledge receipt within 48 hours.
2. Triage and validate the report.
3. Fix and release patch quickly.
4. Notify reporter when fixed.

## Disclosures

- Security issues will be disclosed publicly after a fix is released.
- Coordinated disclosure is preferred.
